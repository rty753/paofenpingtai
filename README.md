```bash
npm start
```

## 🔧 部署指南

### 🚀 Vercel 部署（推荐）

1. **推送到GitHub**
```bash
git add .
git commit -m "Initial commit"
git push origin main
```

2. **Vercel导入**
- 访问 [vercel.com](https://vercel.com)
- 导入GitHub项目
- 自动部署完成

3. **自定义域名**
- 在Vercel设置中添加域名
- 配置DNS记录
- 启用HTTPS

### ☁️ Cloudflare 配置

1. **DNS设置**
```
Type: CNAME
Name: @
Target: your-project.vercel.app
```

2. **SSL配置**
- 启用 "Full (strict)" SSL
- 开启 "Always Use HTTPS"
- 配置 HSTS

3. **性能优化**
- 启用 "Auto Minify"
- 开启 "Brotli Compression"  
- 配置缓存规则

### 🔄 域名更新
